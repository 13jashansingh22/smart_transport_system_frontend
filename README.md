# Smart Transport System Monorepo

This repository is organized with separate folders for frontend and backend.

## Folders

- `frontend/` - Flutter client application
- `backend/` - Node.js + Express API starter

## Frontend Quick Start

```bash
cd frontend
flutter pub get
flutter run -t lib/main.dart -d windows
```

## Backend

See `backend/README.md` for setup, run, and test commands.
